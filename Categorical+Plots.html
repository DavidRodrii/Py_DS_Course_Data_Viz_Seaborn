Bar Plot¶

Most basic categorical plot

In [2]:
sns.barplot(x='sex',y='total_bill',data=tips)
Out[2]:
<matplotlib.axes._subplots.AxesSubplot at 0x1168a7cf8>

input arguments: x = 'Categorical Column Name', y = 'Numerical Column Name', data = Data Frame

Gives us the average total bill for both categories

There's also a 4th input argument we can use: estimator = aggregate function

In [3]:
import numpy as np
In [4]:
sns.barplot(x='sex',y='total_bill',data=tips,estimator=np.std)
Out[4]:
<matplotlib.axes._subplots.AxesSubplot at 0x1169132b0>

Now we can observe the standard deviation between the two categories
