PairGrid¶

Gives us empty pair grid from the pair plots above

In [17]:
sns.PairGrid(iris)
Out[17]:
<seaborn.axisgrid.PairGrid at 0x11f670048>

Main way to use it: set this pair grid equal to a variable, then off of that variable we can map a plot type of our own choosing with the .map(plt.scatter) function

In [18]:
g = sns.PairGrid(iris)
g.map_diag(sns.distplot)
g.map_upper(plt.scatter)
g.map_lower(sns.kdeplot)
